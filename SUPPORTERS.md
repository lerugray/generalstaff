# Supporters

Per Hard Rule 10, GeneralStaff has no company layer. Support goes to
Ray personally through [GitHub Sponsors](https://github.com/sponsors/lerugray).

Every hour sponsored is an hour this repo gets worked on instead of
the minimum-wage day job. There are no recurring obligations, no
supporter tiers, no gated features — the public repo is the same
code a sponsor would be helping fund.

## Sponsors

*Empty at launch. Sponsors who want to be named will be listed here
